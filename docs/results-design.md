# 🍪 두바이 쫀득 쿠키 테스트: 32종 결과 설계 문서

## 📌 기획 배경

**'두바이 초콜릿'** 트렌드를 반영하여 **[피스타치오 + 카다이프]**의 쫀득하고 바삭한 질감을 테마로 설정합니다.

- **키워드 전략**: 대중적인 '두바이' 키워드 활용으로 검색 유입 극대화
- **컨셉**: "~~한 두바이 쫀득 쿠키" 형식으로 32종 결과 제공
- **수익화**: AdSense 광고 최적화 및 바이럴 콘텐츠 확산

## 🎯 분석 알고리즘

### Primary (16종): MBTI 지표

4가지 지표(EI, SN, TF, JP)의 점수 합산으로 기본 MBTI 유형 결정

- **A 유형**: '타인/관계/수용' 성향이 강한 경우
- **B 유형**: '자아/성취/주도' 성향이 강한 경우

## 📊 결과 페이지 필수 데이터 필드

각 결과마다 아래 데이터가 매칭되어야 합니다.

```typescript
interface TestResult {
  // 기본 식별 정보
  id: string; // 예: 'INTJ-A', 'ENFP-B'
  mbti: string; // 예: 'INTJ', 'ENFP'
  variant: 'A' | 'B'; // A: 관계/감성 지향, B: 성취/주도 지향

  // 콘텐츠
  title: string; // 예: "전략적으로 숨겨둔 비상용 두바이 쫀득 쿠키"
  description: string; // 2~3줄 설명 문구
  imageUrl: string; // 결과 이미지 경로

  // 쿠키 특성 점수 (0~100)
  scores: {
    crunch: number; // 바삭함 (카다이프 비중)
    stickiness: number; // 쫀득함 (쿠키 반죽의 찰기)
    nutty: number; // 고소함 (피스타치오 함량)
  };

  // 케미 매칭
  matching: {
    good: string; // 환상의 짝꿍 쿠키 ID (예: 'ENTJ-A')
    bad: string; // 상극인 쿠키 ID (예: 'ISTP-B')
  };
}
```

### 데이터 필드 활용 예시

```typescript
const result: TestResult = {
  id: 'INTJ-B',
  mbti: 'INTJ',
  variant: 'B',
  title: '0.1g의 오차도 없는 황금 비율 두바이 쫀득 쿠키',
  description:
    '카다이프의 바삭함과 피스타치오의 비율을 완벽히 계산한 공학적 맛. 정밀함과 완벽함을 추구하는 당신은 모든 것을 계획대로 실행하는 전략가입니다.',
  imageUrl: '/results/intj-b.png',
  scores: {
    crunch: 85,
    stickiness: 60,
    nutty: 90,
  },
  matching: {
    good: 'ENFP-A',
    bad: 'ESFP-B',
  },
};
```

## 🎨 추가 구현 요소

### 1. 결과 공유 메시지 템플릿

```
나는 "{title}"! 🍪
두바이 쫀득 쿠키 테스트 결과: {mbti}-{variant}

바삭함 {crunch}% | 쫀득함 {stickiness}% | 고소함 {nutty}%

너도 테스트해봐! 👉 [링크]
```

### 2. 점수별 비주얼 표현

- **Progress Bar**: 3가지 특성 점수를 시각적으로 표현
- **Radar Chart**: 전체 밸런스를 한눈에 파악
- **Badge**: 특정 점수가 90점 이상일 경우 특별 뱃지 부여

### 3. 케미 매칭 로직

- **Good Match**: 보완적 성향 (예: I ↔ E, T ↔ F)
- **Bad Match**: 충돌 가능성 높은 조합 (예: J-B ↔ P-A)
